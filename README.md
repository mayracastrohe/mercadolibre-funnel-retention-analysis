# MercadoLibre Funnel & Retention Analysis

## 📌 Project Overview
This project analyzes user behavior through the conversion funnel and retention patterns for an e-commerce platform (MercadoLibre).

The objective is to understand how users move through key stages and identify opportunities to improve retention and engagement.

---

## 📊 Business Questions
- How do users behave across the conversion funnel?
- Which cohorts show the highest and lowest retention?
- What actions can improve user retention?

---

## 🛠 Tools & Technologies
- SQL
- Cohort Analysis
- Funnel Analysis

---

## 🔍 Key Tasks
- Data extraction and analysis using SQL queries  
- Funnel analysis to identify drop-off points  
- Cohort analysis to evaluate retention over time  
- Identification of high and low-performing user segments  

---

## 📈 Key Findings
- Certain cohorts show significantly higher retention rates than others  
- Drop-off points identified in early stages of the funnel  
- User engagement varies depending on acquisition period  

---

## 🚀 Business Implications
- Improve onboarding process to reduce early drop-off  
- Implement targeted notifications to increase engagement  
- Optimize retention strategies for low-performing cohorts
